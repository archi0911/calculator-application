# 🧮 Basic Calculator Web Application

A simple and interactive Calculator Application built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## 📌 Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear input
- Evaluate expressions
- Simple UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Clone the repository:

git clone https://github.com/your-username/calculator-app.git

2. Go to project folder:

cd calculator-app

3. Open index.html in your browser

## ⚙️ How It Works

- Click buttons to enter numbers and operators
- Values are added to the input field
- "=" button evaluates the expression using JavaScript eval()

## ⚠️ Note

This project uses eval() for simplicity.  
It is not recommended for production use due to security risks.
